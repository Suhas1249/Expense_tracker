**MONEY MIND — Personal Expense Tracker**

Track your expenses effortlessly. A clean, modern, secure expense tracking web app with user authentication, budget management, dark mode, and a beautiful responsive dashboard.

**Features**

User Authentication with Gmail validation, Sign up and login securely, Add transactions with description amount category and type, Categorize spending across Food Transport Shopping Bills Entertainment Health and Other, Real time balance updates showing total income and expenses, Monthly budget setting and tracking with visual progress bar, Dark mode toggle for comfortable viewing, Persistent data storage using browser local storage per user account, Delete individual transactions or clear all data, Responsive design works on mobile and desktop, Clean intuitive user interface built with Tailwind CSS.

**Tech Stack**

HTML5 for page structure and semantics, Tailwind CSS for modern responsive styling, Vanilla JavaScript for all app logic and interactivity, Font Awesome for beautiful icons, Local Storage API for data persistence, No backend or database required.

**Project Structure**

index dot html is the landing page with app intro and feature highlights, auth dot html handles user signup and login with Gmail validation, dashboard dot html is the main app with transaction management and budget tracking.

**Installation**

Clone or download this repository, Open index dot html in a modern web browser, No server setup or dependencies needed, All data is stored locally in your browser.

**How to Use**

Open index dot html to see the landing page, Click Start Tracking Now to go to auth dot html, Create a new account with your Gmail address and password minimum six characters, Login with your credentials, On the dashboard add transactions by entering description selecting amount and type income or expense choosing a category and clicking Add Transaction, View all your transactions in the Recent Transactions list, See your total balance income and expenses update in real time, Set a monthly budget by entering amount and clicking Save to track spending progress, Toggle dark mode using the moon icon, Delete individual transactions by clicking Delete or clear all data with Clear All button, Click Logout when done.

**Features in Detail**

Authentication only Gmail dot com emails allowed, Password minimum six characters required, Each user has separate transaction data, Transactions each transaction includes description amount type income or expense category and date, Balance calculation automatically calculates total balance income minus expenses, Budget tracking shows monthly spending against your budget with a progress bar, Dark mode preference is remembered in local storage, Responsive design adapts to mobile tablet and desktop screens.

**Data Storage**

All data stored in browser local storage, User accounts stored as users array, Transactions stored per user as transactions user at email, Budget stored per user as budget user at email, Dark mode preference stored as darkMode, No data sent to any server.

**Browser Compatibility**

Works on Chrome Firefox Edge Safari and any modern browser supporting local storage and ES6 JavaScript.

**Future Enhancement Ideas**

Add expense charts and analytics, Export transactions to CSV or PDF, Set category based budgets, Recurring transactions, Multi currency support, Cloud backup with backend database, Mobile app version.
